# Python Functions

## Count Down
- **File Name**: `count_down.py`
- **Function Name**: `countDown`
- **Description**: Print numbers from a given positive integer down to 0.
- **Run**: `python main.py` or directly `cd modules && python count_down.py`.

## Count Up
- **File Name**: `count_up.py`
- **Function Name**: `countUp`
- **Description**: Print numbers from 0 up to a given positive integer.
- **Run**: `python main.py` or directly `cd modules && python count_up.py`.

## First Uppercase Character
- **File Name**: `first_uppercase.py`
- **Function Name**: `firstUppercaseCharacter`
- **Description**: Find and print the first uppercase character in a string.
- **Run**: `python main.py` or directly `cd modules && python first_uppercase.py`.

## Find Min Max Element
- **File Name**: `min_max_element.py`
- **Function Name**: `findMinMax`
- **Description**: Find and print the minimum and maximum elements in a list.
- **Additional Information**: There is a `documents` folder containing a file named `sample.txt`. The script reads from this file to find and print the first uppercase character.

- **Run**: `python main.py` or directly `cd modules && python min_max_element.py`.

## Print List Elements
- **File Name**: `print_list.py`
- **Function Name**: `printListElements`
- **Description**: Print elements of a list.
- **Run**: `python main.py` or directly `cd modules && python print_list.py`.

## Sum of Digits
